# O Que Ainda Doi - Ecos da Harmonia

Clean folder map for the book project.

## Upload-ready files

Use `amazon-kdp/` for Amazon KDP uploads.

### English

- `amazon-kdp/english/ebook/` - Kindle EPUB upload.
- `amazon-kdp/english/paperback/` - paperback interior PDF upload.
- `amazon-kdp/english/cover/` - front cover image. A full wraparound paperback cover still needs to be generated in KDP after confirming page count, paper, ink, and trim size.
- `amazon-kdp/english/metadata/` - title, description, categories, keywords, pricing notes.

### Portuguese

- `amazon-kdp/portuguese/ebook/` - Kindle EPUB upload.
- `amazon-kdp/portuguese/paperback/` - paperback interior PDF upload.
- `amazon-kdp/portuguese/cover/` - front cover images. A full wraparound paperback cover still needs to be generated in KDP after confirming page count, paper, ink, and trim size.
- `amazon-kdp/portuguese/metadata/` - title, description, categories, keywords, pricing notes.

## Reading PDFs

Use `reading-pdfs/` for normal reading/review PDFs with cover included.

- `reading-pdfs/english/`
- `reading-pdfs/portuguese/`

## Source and working files

- `06-capitulos/` - Portuguese chapters.
- `05-ficheiros-proibidos/` - Portuguese file/interlude sections.
- `08-manuscrito/` - compiled Portuguese manuscript.
- `09-english/` - English adaptation source and compiled manuscript.
- `images/` - book cover/image assets.
- `scripts/` - build scripts.
- `output/` - raw generated build output kept for regeneration/debugging.
- `site-staging/` - local site staging and screenshots.
- `tmp/` - temporary QA/rendering files.
