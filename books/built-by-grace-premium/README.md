# Built by Grace — Premium Edition

This folder contains a separate premium editorial edition. It does not overwrite the currently published files.

## Editorial principles

- Narrative reflection is set as readable prose rather than line-broken verse.
- Prayers, blessings, litanies, and deliberately rhythmic endings remain spacious.
- Repetition is retained when it works as spiritual emphasis and reduced when it merely restates an idea.
- The authorial voice remains tender, direct, prayerful, and accessible.
- Each chapter keeps the distinct emotional color defined in `editorial-direction.md`.

## Files

- `build/built-by-grace-source-manuscript.md`: untouched source snapshot.
- `build/built-by-grace-premium-manuscript.md`: premium editorial manuscript.
- `editorial-direction.md`: chapter-by-chapter rhythm guidance.
- `scripts/reflow-premium-manuscript.mjs`: reproducible prose-reflow pass.
